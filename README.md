Wheat and Rice Distribution Analysis (Power BI)

## Project Overview
This project analyzes the distribution of wheat and rice across different states and districts in India using Power BI.  
The dashboard helps understand allocation, distribution, and balance stock of food grains.

## Objectives
- Analyze wheat and rice allocation vs distribution
- Identify states with highest distribution
- Find balance stock of wheat and rice
- Compare distribution by state, district, and month

## Dataset
The dataset contains:
- State Name
- District Name
- Month
- Total Rice Allocated
- Total Rice Distributed
- Total Wheat Allocated
- Total Wheat Distributed
- Balance Rice
- Balance Wheat

## Tools Used
- Power BI Desktop
- Microsoft Excel
- Data Cleaning
- Data Visualization

## Dashboard Features
The dashboard includes:
- State-wise distribution of wheat and rice
- District-wise rice allocation vs distribution
- Monthly distribution trends
- Balance wheat and rice analysis

## Key Insights
- Rice distribution is higher than wheat distribution.
- Allocation is usually slightly higher than distribution, leaving some balance stock.
- Madhya Pradesh and Maharashtra have the highest wheat balances.
- Maharashtra, Uttar Pradesh, and Bihar show the highest rice balances.
- Some districts show zero wheat distribution.
